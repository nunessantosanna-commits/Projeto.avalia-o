Visão Geral e termo de abertura
Grand Le Palais é um layout de alta fidelidade desenvolvido no Figma focado na experiência de reserva de quartos para hotéis de 5 estrelas. O design combina sofisticação visual com uma jornada de usuário minimalista e intuitiva. O objetivo principal é transformar visitantes em hóspedes, transmitindo exclusividade desde o primeiro clique.
